We will give you a quick introduction before you start submitting a model to the MMHCC Cancer Models Database.
Alternatively, you can take the "Guided Tour" or refer to the Help Documentation.

For each new model it is necessary to complete the information on the "General Information page". Required fields on this page are:
  • Model Descriptor
  • PI's Email Address
  • Species of the Model
  • Phenotype
  • Record Release Date
After you are have submitted these data the program will open the "Main page" which contains links to all parts of the submission process.

You have to fill out only the parts that apply to your model, e.g. if your model was not exposed to carcinogenic agents you don't have to enter data about carcinogenic agents.

You can make multiple entries for each category, e.g. if the model was exposed to two different chemicals or if the model was used in multiple therapeutic trials.

If you cannot complete the submission of the model in one session you can come back later and add data.

Help Several Help features are build into this application.
We have two Quick Help options and extended Help in the Help file.

Quick Help Option 1 - Move your mouse pointer over the help icon. A message will appear in the bottom part of your browser.
Quick Help Option 2 - Move your mouse pointer over the help icon and hold it there for a second or two. A tool tip message will appear next to your mouse pointer.

Click on the help icon and the Help file will be displayed. The help file will open in a new browser window. You'll need to close the Help window in order to continue the submission process.
